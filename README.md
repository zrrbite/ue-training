# Unreal Engine Training Program

This repository contains practical examples and exercises for mastering Unreal Engine C++ concepts.

## Training Modules

### Module 1: Smart Pointers and References in Unreal Engine
**Status:** In Progress

Topics covered:
- UObject pointers and garbage collection
- TSharedPtr, TSharedRef, TWeakPtr
- TUniquePtr
- When to use each type
- Common pitfalls and best practices

**Location:** `Module01_SmartPointers/`

### Module 2: Task System (TTask)
**Status:** Planned

Topics covered:
- TTask basics and use cases
- Task graph system
- Async task execution
- Task dependencies and chaining
- Performance considerations

**Location:** `Module02_TaskSystem/`

## How to Use This Repository

Each module contains:
- **README.md** - Theory, explanations, and guidelines
- **Examples/** - Working code demonstrating concepts
- **Exercises/** - Practice problems to reinforce learning

Work through modules sequentially, as later modules may build on earlier concepts.
